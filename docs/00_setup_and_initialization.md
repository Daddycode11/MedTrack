# 00. Setup and Initialization

This section provides a comprehensive guide to setting up and initializing the project. It covers the prerequisites, installation steps, and configuration options necessary to get started.

# 1.1 Prerequisites
Before you begin, ensure you have the following installed on your machine:
- Python 3.11 or higher
- git

# 1.2 Installation
To install the project, follow the steps in the README file.

# 1.3 Configuration and initialization

After installation, you may need to configure the project to suit your needs. However, the default configuration should work out of the box.

Initialize the project by running the following command in your terminal:
```bash
python manage.py initialize
```

This command will set up the necessary database tables and create the initial data.
